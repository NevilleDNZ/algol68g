Algol68G extends the standard environ as described by the Revised Report with a number of procedures and constants.

Many of these extensions require the GNU Scientific Library (GSL, version 1.8 or compatible). This library is optional; if it is not present on your system, Algol68G will build without support for routines and constants that depend on this library.

This section contains a list of procedures supported by the current revision of Algol68G. Items that require GSL are marked as [GSL]. Items that do not require GSL are marked as [Native].

REAL functions

PROC sinh = (REAL x) REAL
PROC long sinh = (LONG REAL x) LONG REAL
PROC long long sinh = (LONG LONG REAL x) LONG LONG REAL
[Native] Evaluates the hyperbolic sine of argument x.

PROC arc sinh = (REAL x) REAL
PROC long arc sinh = (LONG REAL x) LONG REAL
PROC long long arc sinh = (LONG LONG REAL x) LONG LONG REAL
[Native] Evaluates the inverse hyperbolic sine of argument x.

PROC cosh = (REAL x) REAL
PROC long cosh = (LONG REAL x) LONG REAL
PROC long long cosh = (LONG LONG REAL x) LONG LONG REAL

PROC arc cosh = (REAL x) REAL

PROC long long tanh = (LONG LONG REAL x) LONG LONG REAL
[Native] Evaluates the hyperbolic tangent of argument x.

PROC arc tanh = (REAL x) REAL
PROC long arc tanh = (LONG REAL x) LONG REAL

REAL Bessel functions

PROC bessel jn = (REAL n, REAL x) REAL
[GSL] Evaluates the regular cylindrical Bessel function J_n (x). Argument n must be integral.

PROC bessel yn = (REAL n, REAL x) REAL
[GSL] Evaluates the irregular cylindrical Bessel function Y_n (x). Argument n must be integral.

PROC bessel in = (REAL n, REAL x) REAL
[GSL] Evaluates the regular modified cylindrical Bessel function I_n (x). Argument n must be integral.

PROC bessel exp in = (REAL n, REAL x) REAL
[GSL] Evaluates the scaled regular modified cylindrical Bessel function exp (-|x|) I_n (x). Argument n must be integral.

PROC bessel kn = (REAL n, REAL x) REAL
[GSL] Evaluates the irregular modified cylindrical Bessel function K_n (x). Argument n must be integral.

PROC bessel exp kn = (REAL n, REAL x) REAL
[GSL] Evaluates the scaled regular modified cylindrical Bessel function exp (-|x|) K_n (x). Argument n must be integral.

PROC bessel jl = (REAL l, REAL x) REAL
[GSL] Evaluates the regular spherical Bessel function j_l (x). Argument l must be integral.

PROC bessel yl = (REAL l, REAL x) REAL
[GSL] Evaluates the irregular spherical Bessel function y_l (x). Argument l must be integral.

PROC bessel exp il = (REAL n, REAL x) REAL
[GSL] Evaluates the scaled regular modified spherical Bessel function exp (- |x|) i_l (x). Argument l must be integral.

PROC bessel exp kl = (REAL n, REAL x) REAL
[GSL] Evaluates the scaled irregular modified spherical Bessel function exp (-|x|) k_l (x). Argument l must be integral.

PROC bessel jnu = (REAL nu, REAL x) REAL
[GSL] Evaluates the fractional regular cylindrical Bessel function J_nu (x).

PROC bessel ynu = (REAL nu, REAL x) REAL
[GSL] Evaluates the fractional irregular cylindrical Bessel function Y_nu (x).

PROC bessel inu = (REAL nu, REAL x) REAL
[GSL] Evaluates the fractional regular modified Bessel function J_nu (x).

PROC bessel exp inu = (REAL nu, REAL x) REAL
[GSL] Evaluates the scaled fractional regular modified Bessel function exp (- |x|) I_nu (x).

PROC bessel knu = (REAL nu, REAL x) REAL
[GSL] Evaluates the fractional irregular modified Bessel function Y_nu (x).

PROC bessel exp knu = (REAL nu, REAL x) REAL
[GSL] Evaluates the scaled fractional irregular modified Bessel function exp (-|x|) K_nu (x).

REAL Elliptic integrals

[GSL] Evaluates the complete elliptic integral of the first kind K (x).

PROC elliptic integral e = (REAL x) REAL
[GSL] Evaluates the complete elliptic integral of the second kind E (x).

PROC elliptic integral rf = (REAL x, y, z) REAL
[GSL] Evaluates Carlson's elliptic integral of the first kind R_F (x, y, z).

PROC elliptic integral rd = (REAL x, y, z) REAL
[GSL] Evaluates Carlson's elliptic integral of the second kind R_D (x, y, z).

PROC elliptic integral rj = (REAL x, y, z, rho) REAL
[GSL] Evaluates Carlson's elliptic integral of the third kind R_J (x, y, z, rho).

PROC elliptic integral rc = (REAL x, y) REAL
[GSL] Evaluates Carlson's elliptic integral of the third kind R_C (x, y).

REAL Error and Gamma functions

PROC erf = (REAL x) REAL
[GSL] Evaluates the error function of argument x.

PROC inverse erf = (REAL x) REAL
[Native] Evaluates the inverse error function of argument x.

PROC erfc = (REAL x) REAL
[GSL] Evaluates the complementary error function of argument x, defined as 1 - erf (x).

PROC inverse erfc = (REAL x) REAL
[Native] Evaluates the inverse complementary error function of argument x.

PROC gamma = (REAL x) REAL
[GSL] Evaluates the gamma function of argument x.

PROC incomplete gamma = (REAL a, x) REAL
[GSL] Evaluates the incomplete gamma function P (a, x).

PROC ln gamma = (REAL x) REAL
[GSL] Evaluates the natural logarithm of the gamma function of argument x.

PROC factorial = (REAL x) REAL
[GSL] Evaluates the factorial of argument x, defined as gamma (1 + x).

PROC beta = (REAL a, b) REAL
[GSL] Evaluates the beta function of arguments a and b, defined as gamma (a) * gamma (b) / gamma (a + b).

PROC incomplete beta = (REAL a, b, x) REAL
[GSL] Evaluates the normalised incomplete beta function of arguments a, b and x.
