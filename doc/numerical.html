On systems that have installed the GNU Scientific Library, Algol68G can be linked with that library to extend the standard prelude. This section contains a list of procedures supported by the current revision of Algol68G. Some listed items are not part of GSL and are marked as [non-GSL].


REAL functions

PROC sinh = (REAL x) REAL
[non-GSL] Evaluates the hyperbolic sine of argument x.

PROC arc sinh = (REAL x) REAL
Evaluates the inverse hyperbolic sine of argument x.

PROC cosh = (REAL x) REAL
[non-GSL] Evaluates the hyperbolic cosine of argument x.

PROC arc cosh = (REAL x) REAL
Evaluates the inverse hyperbolic cosine of argument x.

PROC tanh = (REAL x) REAL
[non-GSL] Evaluates the hyperbolic tangent of argument x.

PROC arc tanh = (REAL x) REAL
Evaluates the inverse hyperbolic tangent of argument x.

REAL Airy functions

PROC airy ai = (REAL x) REAL
Evaluates the airy function Ai (x).

PROC airy bi = (REAL x) REAL
Evaluates the airy function Bi (x).

PROC airy ai derivative = (REAL x) REAL
Evaluates the airy function Ai' (x).

PROC airy bi derivative = (REAL x) REAL
Evaluates the airy function Bi' (x).

REAL Bessel functions

PROC bessel jn = (REAL n, REAL x) REAL
Evaluates the regular cylindrical Bessel function J_n (x). Argument n must be integral.

PROC bessel yn = (REAL n, REAL x) REAL
Evaluates the irregular cylindrical Bessel function Y_n (x). Argument n must be integral.

PROC bessel in = (REAL n, REAL x) REAL
Evaluates the regular modified cylindrical Bessel function I_n (x). Argument n must be integral.

PROC bessel exp in = (REAL n, REAL x) REAL
Evaluates the scaled regular modified cylindrical Bessel function exp (-|x|) I_n (x). Argument n must be integral.

PROC bessel kn = (REAL n, REAL x) REAL
Evaluates the irregular modified cylindrical Bessel function K_n (x). Argument n must be integral.

PROC bessel exp kn = (REAL n, REAL x) REAL
Evaluates the scaled regular modified cylindrical Bessel function exp (-|x|) K_n (x). Argument n must be integral.

PROC bessel jl = (REAL l, REAL x) REAL
Evaluates the regular spherical Bessel function j_l (x). Argument l must be integral.

PROC bessel yl = (REAL l, REAL x) REAL
Evaluates the irregular spherical Bessel function y_l (x). Argument l must be integral.

PROC bessel exp il = (REAL n, REAL x) REAL
Evaluates the scaled regular modified spherical Bessel function exp (- |x|) i_l (x). Argument l must be integral.

PROC bessel exp kl = (REAL n, REAL x) REAL
Evaluates the scaled irregular modified spherical Bessel function exp (-|x|) k_l (x). Argument l must be integral.

PROC bessel jnu = (REAL nu, REAL x) REAL
Evaluates the fractional regular cylindrical Bessel function J_nu (x).

PROC bessel ynu = (REAL nu, REAL x) REAL
Evaluates the fractional irregular cylindrical Bessel function Y_nu (x).

PROC bessel inu = (REAL nu, REAL x) REAL
Evaluates the fractional regular modified Bessel function J_nu (x).

PROC bessel exp inu = (REAL nu, REAL x) REAL
Evaluates the scaled fractional regular modified Bessel function exp (- |x|) I_nu (x).

PROC bessel knu = (REAL nu, REAL x) REAL
Evaluates the fractional irregular modified Bessel function Y_nu (x).

PROC bessel exp knu = (REAL nu, REAL x) REAL
Evaluates the scaled fractional irregular modified Bessel function exp (-|x|) K_nu (x).

REAL Elliptic integrals

PROC elliptic integral k = (REAL x) REAL
Evaluates the complete elliptic integral of the first kind K (x).

PROC elliptic integral e = (REAL x) REAL
Evaluates the complete elliptic integral of the second kind E (x).

PROC elliptic integral rf = (REAL x, y, z) REAL
Evaluates Carlson's elliptic integral of the first kind R_F (x, y, z).

PROC elliptic integral rd = (REAL x, y, z) REAL
Evaluates Carlson's elliptic integral of the second kind R_D (x, y, z).

PROC elliptic integral rj = (REAL x, y, z, rho) REAL
Evaluates Carlson's elliptic integral of the third kind R_J (x, y, z, rho).

PROC elliptic integral rc = (REAL x, y) REAL
Evaluates Carlson's elliptic integral of the third kind R_C (x, y).

REAL Error and Gamma functions

PROC erf = (REAL x) REAL
Evaluates the error function of argument x.

PROC inverse erf = (REAL x) REAL
[non-GSL] Evaluates the inverse error function of argument x.

PROC erfc = (REAL x) REAL
Evaluates the complementary error function of argument x, defined as 1 - erf (x).

PROC inverse erfc = (REAL x) REAL
[non-GSL] Evaluates the inverse complementary error function of argument x.

PROC gamma = (REAL x) REAL
Evaluates the gamma function of argument x.

PROC incomplete gamma = (REAL a, x) REAL
Evaluates the incomplete gamma function P (a, x).

PROC ln gamma = (REAL x) REAL
Evaluates the natural logarithm of the gamma function of argument x.

PROC factorial = (REAL x) REAL
Evaluates the factorial of argument x, defined as gamma (1 + x).

PROC beta = (REAL a, b) REAL
Evaluates the beta function of arguments a and b, defined as gamma (a) * gamma (b) / gamma (a + b).

PROC incomplete beta = (REAL a, b, x) REAL
Evaluates the normalised incomplete beta function of arguments a, b and x.

COMPLEX functions

PROC complex sqrt = (COMPLEX z) COMPLEX
Evaluates the complex square root of argument z. The branch cut is the negative real axis. The result lies in the right half of the complex plane.

PROC complex exp = (COMPLEX z) COMPLEX
Evaluates the complex exponential of argument z.

PROC complex ln = (COMPLEX z) COMPLEX
Evaluates the complex natural logarithm of argument z. The branch cut is the negative real axis.

PROC complex sin = (COMPLEX z) COMPLEX
Evaluates the complex sine of argument z.

PROC complex arc sin = (COMPLEX z) COMPLEX
Evaluates the complex inverse sine of argument z. The branch cuts are on the real axis, less than -1 and greater than 1.

PROC complex cos = (COMPLEX z) COMPLEX
Evaluates the complex cosine of argument z.

PROC complex arc cos = (COMPLEX z) COMPLEX
Evaluates the complex inverse cosine of argument z. The branch cuts are on the real axis, less than -1 and greater than 1.

PROC complex tan = (COMPLEX z) COMPLEX
Evaluates the complex tangent of argument z.

PROC complex arc tan = (COMPLEX z) COMPLEX
Evaluates the complex inverse tangent of argument z. The branch cuts are on the imaginary axis, below -i and above i.

PROC complex sinh = (COMPLEX z) COMPLEX
Evaluates the complex hyperbolic sine of argument z.

PROC complex arc sinh = (COMPLEX z) COMPLEX
Evaluates the complex inverse hyperbolic sine of argument z. The branch cuts are on the imaginary axis, below -i and above i.

PROC complex cosh = (COMPLEX z) COMPLEX
Evaluates the complex hyperbolic cosine of argument z.

PROC complex arc cosh = (COMPLEX z) COMPLEX
Evaluates the complex inverse hyperbolic cosine of argument z. The branch cuts are on the real axis, less than -1 and greater than 1.

PROC complex tanh = (COMPLEX z) COMPLEX
Evaluates the complex hyperbolic tangent of argument z.

PROC complex arc tanh = (COMPLEX z) COMPLEX
Evaluates the complex inverse hyperbolic tangent of argument z. The branch cuts are on the real axis, less than -1 and greater than 1.
