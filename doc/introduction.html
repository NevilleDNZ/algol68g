What is Algol 68 Genie?

Algol 68 is a rather lean, orthogonal general-purpose programming language, that is a beautiful means for denoting algorithms.

Algol 68 Genie (Algol68G) is an Algol 68 interpreter. It can be used for executing Algol 68 programs or scripts when execution time is not of primary importance; Algol68G is pedantic rather than fast.

Algol68G is free software under the GNU General Public License. The current version is Mark 10, released in August 2006. You can download it from this page.

  • Download Algol68G Mark 10, a gzipped tar archive containing C source code and documentation.

    Here are instructions for installation of Algol68G on UNIX, Linux or MacOS X.

    The archive includes a68g.exe, a WIN32 executable of Algol 68 Genie Mark 10, which has been built and tested on Microsoft Windows XP and can be run from cmd.exe. This executable has been linked to the GNU Plotting Utilities and the GNU Scientific Library.

    Algol68G Mark 10 is regression-tested using a test set of more than 1100 programs.

  • For Apple's MacOS X Darwin environment, Norman Hardy has posted a binary package.
  • The resources page contains links to on-line Algol 68 resources (Revised Report, informal textbook, historic documents, Algol Bulletin archive, other implementations, and more). The example programs page has examples of Algol68G programs.

What libraries can Algol 68 Genie link to?

  • GNU Plotutils for drawing from Algol 68.
  • GNU Scientific Library to extend the mathematical capabilities of the standard environ.
  • PostgreSQL for writing PostgreSQL database client applications.
