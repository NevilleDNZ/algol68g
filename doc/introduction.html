What is Algol 68 Genie?

Algol 68 is a rather lean, orthogonal general-purpose programming language, that even today is a beautiful means for denoting algorithms.

Algol 68 Genie (Algol68G) is an Algol 68 interpreter. It can be used for executing Algol 68 programs or scripts when execution time is not of primary importance; Algol68G is pedantic rather than fast.

Algol68G is free software under the GNU General Public License. The current version is Mark 8, released in July 2005. You can download it from this page.


Algol 68 Genie features

Algol 68 Genie Mark 8 features:

  • A check-out interpreter with many runtime checks, for example uninitialised values, overflow, accessing NIL, dynamic scope ("dangling references") and bounds.
  • A front-end translator that parses program text and builds a syntax tree. The back-end interpretes the syntax tree.
  • Multiprecision modes:LONG and (with user defined precision) LONG LONG precisions for modes INT, REAL, COMPLEX and BITS.
  • Implementation of C.H. Lindsey's partial parametrisation proposal, which is an extension of Algol 68. Partial parametrisation allows for currying in Algol 68, giving it a functional sublanguage [Koster 1996].
  • Optional graphical procedures using the GNU Plotting Utilities, for drawing in X Windows, postscript format, pseudo graphical interface format (pseudo-gif) and portable any-map format (pnm) from Algol 68.
  • On systems that support them, UNIX extensions that allow e.g. for executing child processes that communicate through pipes, matching regular expressions or fetching web page contents.
  • Format texts, straightening and formatted transput. Transput procedures operate generically on files, strings and UNIX pipes.
  • Parallel clause on platforms that support POSIX threads (e.g. Linux, UNIX).
  • Elementary and trigonometric functions for complex numbers independent of GNU Scientific Library.
  • Optional numerical procedures using the GNU Scientific Library.


Obtaining Algol 68 Genie

Algol 68 Genie (Algol68G) is free software, distributed under GNU General Public License. This software is distributed in the hope that it will be useful, but without any warranty; without even the implied warranty of merchantability or fitness for a particular purpose. Consult the GNU General Public License for details.

  • Download Algol68G Mark 8, a gzipped tar archive containing C source code and documentation.
    Algol68G Mark 8 is regression-tested using a test set of more than 1000 programs.
    Here are instructions for installation of Algol68G on GNU/Linux.
  • Download an Algol68G Mark 7 Win32 executable for Microsoft Windows, which is made available by Theo Vosse. This gzipped tar archive contains the executable and documentation. Unzip (using f.i. WinZip) the archive to obtain a68g.exe which can be executed from command prompts command.com or cmd.exe. The executable is regression-tested and linked to the GNU Plotting Utilities. The Algol68G Mark 7 gzipped tar archive is still available.
  • For Apple's MacOS X Darwin environment, Norman Hardy has posted a binary package.
On-line books on Algol 68 from other sites:
  • Revised Report. The formal defining document, in pdf format. Made available by Wilhelm Kloke.
  • Programming Algol 68 Made Easy. An informal textbook by Sian Leitch, in pdf format. Made available by Roy Fullmer.

The resources page contains more links to on-line Algol 68 resources (historic documents, Algol Bulletin archive, other implementations, and more). The example programs page has examples of Algol68G programs.
